2010-05-13 · 237 words · 2 minute read
blog
Mercurial · RHEL5 · python · linux

Installing Mercurial 1.5.1 on Red Hat Enterprise Linux 5 (RHEL5) using Python 2.5 My RHEL5 setup uses python2.5 which I compiled from source, since RHEL5 only comes with python2.4. Because I didn’t use the built in python 2.4 version I couldn’t do the simple $ yum install mercurial Instead I need to build mercurial from source, which isn’t too bad. Just do the following. as Root $ wget http://mercurial.selenic.com/release/mercurial-1.5.1.tar.gz $ tar -xvzf mercurial-1.5.1.tar.gz $ cd mercurial-1.5.1 $ make PYTHON=python2.5 install If you get the following error: $ python2.5 setup.py build $ Couldn’t import standard bz2 (incomplete Python install).
